Professional profile

I started to code in middle school and since then I never stopped doing it. In 2007 I graduated from computer engineering and I've spent my time working on several computing platforms and programming languages mostly developing financial and credit solutions.

My passion is execution optimization (with care, obviously) and concurrent programming, I'm always speeding up some database queries, chasing deadlocks on parallel programs, designing REST APIs, high performance websites and lately squeezing some CPU cycles on assembly language.

I consider myself a backend developer but I've done programs for every layer in lots of projects. If I were you I'd think twice about asking me some web development, but I can give a try and you won't regret (hopefully).

My favorite programming languages are Python, Go and Haskell, but in other lives I worked on Java and C++
